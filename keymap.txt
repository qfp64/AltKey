# Comments start with a #.

# The general format is:
#   glyph [whitespace] key-sequence
# where key-sequence represents a pair of keys.
# For example,
#   á   a3
# Assuming the hotkey is AltGr, pressing AltGr+a and then 3 will generate á.

# Rules can be grouped with []:
#   [áéíóú]   [aeiou]3
# Pressing AltGr plus any of [aeiou] and then 3 will generate the corresponding glyph.
